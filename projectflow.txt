1. Create git repo and clone it in local. ----------> Done
2. Create mycode.py and add code to it. (it will save a csv file to a new "data" folder) ------> Done
3. Do a git add-commit-push before initializing dvc. -------> Done
# pip3 install dvc -------> Done
4. Now we do "dvc init" (creates .dvcignore, .dvc) -------> Done
5. Now do "mkdir S3" (creates a new S3 directory) -------> Done
6. Now we do "dvc remote add -d myremote S3" -------> Done
7. Next "dvc add data/" 
   Now it will ask to do: ("git rm -r --cached 'data'" and "git commit -m "stop tracking data"")
   Because initially we were tracking data/ folder from git so now we remove it for DVC to handle.
8. Again we do "dvc add data/" (creates data.dvc) then "git add data.dvc .gitignore"  
9. Do a git add-commit-push to mark this stage as first version of data.
10. Now make changes to mycode.py to append a new row in data, check changes via "dvc status"
11. Again - - "dvc commit" and then "dvc push"
